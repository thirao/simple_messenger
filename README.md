# simple_messenger
tornado websocket simple messenger

# Requirement
- Python3.5
- Tornado4.4.2

you should create settings.py before use.  
  
settings.py sample:
~~~~
COOKIE_SECRET = "__TODO:_GENERATE_YOUR_OWN_RANDOM_VALUE_HERE__"
~~~~
